Private Infrastructure Made Easy

Homelab as a Service is your own personal infrastructure automation tool. Designed to work with other cheap and easy to aquire solutions, this tool gives you a great experience, and provides a small but useful swiss-army knife array of tools to automatically build and manage virtual machines, network infrastructure and containerised workloads. Think of it as a mini cloud.

Getting Started

Installation is as easy as starting up a special bootstrap container. This container will help you through the setup process and supports a number of ways to run HaaS.

To get started, run:

docker run -v /var/run/docker.sock:/var/run/docker.sock -p 3010:3010 homelabaas/haas-bootstrap

Then browse to http://localhost:3010 to step through the installation.
